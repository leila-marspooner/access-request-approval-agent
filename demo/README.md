# Demo

This folder contains public demo guidance for the Access Request & Approval Agent.

## Demo Goals

- Show conversational request intake.
- Show Dataverse as system of record.
- Show Teams Adaptive Card approval.
- Show BYOM CaseSummary as reviewer support.
- Show mocked/simulated fulfilment.
- Show GetStatus status lookup.
- Show SLA reminder and escalation watcher behaviour.
- Show operational visibility through the Model-driven app.

## Demo Safety

Use redacted data only. Do not publish real emails, private approver names, tenant details, environment URLs, secrets, endpoint URLs, private run history, or live Dataverse records.
