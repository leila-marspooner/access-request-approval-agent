# Assets

This folder is for public-safe supporting assets such as diagrams, icons, and exported images.

## Suggested Structure

- `diagrams/`
- `icons/`
- `exports/`

## Rules

- Do not store private source notes here.
- Do not store unredacted screenshots here.
- Do not include tenant-specific configuration, real emails, endpoint URLs, secrets, or private data.
- Prefer clear descriptive filenames, for example `access-request-architecture.png`.

## Planned Assets

- Optional demo thumbnail
- Optional redacted evidence collage
