# Assets

This folder is for public-safe supporting assets for the **Access Request & Approval Agent** portfolio repo.

Use it for diagrams, icons, exported public images, or other visual material that supports the documentation.

## Suggested Structure

```text
assets/
├── diagrams/
├── icons/
└── exports/
```

## Planned Assets

- Public-safe architecture diagram
- Optional demo thumbnail
- Optional redacted evidence collage
- Optional process lifecycle diagram

## Asset Rules

- Do not store private local notes here.
- Do not store unredacted screenshots here.
- Do not include tenant-specific configuration, real email addresses, endpoint URLs, IDs, secrets, tokens, or private account details.
- Use clear filenames such as `access-request-architecture.png` or `flow-lifecycle-overview.png`.

Screenshots should live under `screenshots/` after redaction. Architecture and design assets can live here.
