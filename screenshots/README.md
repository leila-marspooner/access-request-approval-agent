- Copilot Studio request intake

- RBAC/security role evidence
- Field Security Profile evidence
- Dataverse audit history
- Environment variables and Connection references
- Managed solution / Unmanaged solution evidence
- Release checklist evidence

## Redaction Rules

Before publishing, remove or blur:

- Tenant IDs and tenant URLs
- Environment URLs and environment names
- Private endpoint URLs
- API keys, secrets, client secrets, bearer tokens, and passwords
- Client IDs where they identify private configuration
- Real email addresses and private approver names
- Live Dataverse data
- Private Power Automate run history
- Connection reference secrets or connection details

Use screenshot placeholders until safe evidence is ready.
